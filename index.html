Introduction

Vice is a scripting language built around readable syntax, explicit data flow and simple semantics.

Current state: Vice is currently in beta and some systems are temporary stubs. 

main.vice




@entry
fn main() {
    output <- "Vice works.";
}

Installation & Running

Programs are executed through the Vice CLI. 


terminal




cargo run -- run examples/full_test_app.vice
cargo run -- check examples/full_test_app.vice

Loops


loops.vice




for i in 1..5 {
    output <- "Step:", i;
}

let nums => [10, 20, 30];

for n in nums {
    output <- n;
}

mut count => 3;

while (count > 0) {
    output <- count;
    count -= 1;
}

Data Structures


player.vice




data Player {
    id      : int    => 1;
    name    : string => "New Player";
    isAdmin : bool   => false;
}

mut player => Player();

player.name = "Alex";

output <- player.name;
